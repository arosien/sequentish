=== Deduction ‚Üë

[.theory]
stem:[(A |-- B)/(|-- A => B)]

> üîä "if we conclude [.theory]#stem:[|-- A => B]#, +

=== Inference Rules

:Rimp: pass:m[stem:[(Gamma, A |-- B)/(Gamma |-- A => B)R=>]]
:AimpB: pass:m[stem:[|-- A => B]]
:AentB: pass:m[stem:[A |-- B]]
